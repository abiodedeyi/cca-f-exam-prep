# CCA-F Exam Blueprint

**Source of truth:** the official Anthropic *Claude Certified Architect – Foundations Certification Exam Guide* (PDF). This file summarises structure only — for the actual task-statement text, sample questions, and rationales, read the official guide directly. Do not substitute third-party content.

## Exam structure (confirmed from official sources)

- Multiple choice, scenario-based questions
- Each question: one correct answer + three plausible distractors
- The exam draws from a fixed set of scenarios, selecting a subset per sitting
- Scaled scoring (100–1000 range)
- **Passing score: 720 / 1000**
- No penalty for guessing
- 60 questions on the official practice exam (verified from the practice exam UI)

## Six scenarios used by the exam

The exam frames questions inside realistic scenarios. Candidates are expected to reason within each scenario's context. The six scenarios per the official guide:

1. Customer Support Resolution Agent
2. Code Generation with Claude Code
3. Multi-Agent Research System
4. Developer Productivity with Claude
5. Claude Code for Continuous Integration
6. Structured Data Extraction

## Five content domains with weights

| Domain | Weight | Number of task statements |
|--------|--------|---------------------------|
| **D1** Agentic Architecture & Orchestration | **27%** | 7 |
| **D2** Tool Design & MCP Integration | **18%** | 5 |
| **D3** Claude Code Configuration & Workflows | **20%** | 6 |
| **D4** Prompt Engineering & Structured Output | **20%** | 6 |
| **D5** Context Management & Reliability | **15%** | 6 |
| **Total** | **100%** | **30** |

For the exact task-statement text per domain, see the official guide.

## What to focus on (high-leverage)

- **D1 first** — largest weight, biggest spread of task statements, foundational concepts for the rest.
- **D2 + D3 + D4** in any order — comparable weights, complementary topics.
- **D5 last** — smallest weight but full of nuance; benefits from prior domains being locked.

## Sample questions

The official guide includes worked sample questions with answers and rationales. Reference these directly during study — do NOT rely on third-party question banks for ground truth (some encode wrong answers).

## Out-of-scope (per the official guide)

The official guide explicitly excludes certain topics — fine-tuning, model training, custom classifiers, etc. **Any answer option that suggests an out-of-scope mechanism is almost always wrong on this exam.** Use this as an instant rule-out reflex.

## Practice exam

Anthropic provides an official practice exam (linked separately from the guide). Treat the result as a baseline, not a verdict. Map missed topics to specific concepts, re-drill those concepts, and re-test.
